LOLCAT Language

By CR24

Last Updated: November 30, 2021

Skip to Download

LOLCAT Language is a Minecraft language pack that translates all in-game text into LOLCAT speak, making the game more fun and entertaining.

This language is part of Minecraft Java Edition, but it can also be used in Bedrock Edition by installing this resource pack.
